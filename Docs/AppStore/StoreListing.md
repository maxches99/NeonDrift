# NeonDrift

## Positioning

Animated desktop overlay for macOS.

NeonDrift renders a full-screen animated visual layer behind your desktop icons and spaces. It does not replace the system wallpaper file in System Settings. Instead, it creates a live wallpaper effect by drawing a non-interactive desktop overlay for each connected display.

## Short Description

Animated desktop overlay with shader-driven plasma, fractal, and pattern themes.

## Subtitle Ideas

- Live desktop overlay
- Animated desktop visuals
- Shader wallpapers for Mac

## Full Description

NeonDrift brings animated desktop visuals to macOS with a collection of GPU-rendered themes.

The app creates a non-interactive overlay at the desktop layer, so your icons and normal desktop workflow stay usable while the animation runs beneath them. Choose from plasma gradients, fractal zooms, and geometric patterns, then leave the app running to keep the effect active across your displays.

Highlights:
- Live animated desktop overlay for every connected display
- Multiple theme families: plasma, fractals, and patterns
- Theme picker from the app menu
- Hardware-accelerated Metal rendering
- Remembers the last selected theme

## Review Notes

This app does not change the user's system wallpaper in System Settings.

It works by opening a non-interactive borderless window at the desktop layer for each screen and rendering animated content with Metal. The overlay ignores mouse input, stays behind normal apps, and is used only to create a live wallpaper effect.
